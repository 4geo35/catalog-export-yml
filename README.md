### Usage

    generate:catalog-yml {--fileName=}

Поместить команду в расписание, будет генерировать файл в `public`, название папки и файла можно изменить в конфиге.

Стандартный путь к файлу: `/storage/yml/yandex-products.yml`

Адрес по которому возвращается закешированный на сутки файл (т.е. генерация файла по запросу, а не по расписанию): `/catalog-export/yml`
